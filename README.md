This AI Powered Study Assistant is built during our 24 - hour national level hackathon.

Problem Statement:
Create an AI-powered study assistant that analyzes learning patterns, recommends study materials, generates practice questions, and provides personalized study schedules based on exam dates and course complexity.

🚀 Core Features

- 🤖 **AI Tutor** — Ask doubts, get instant explanations
- ⚡ **Question Generator** — AI creates MCQs for your exact topic
- 📅 **Study Schedule** — Day-by-day plan based on your exam date
- 📊 **Progress Dashboard** — Completion %, scores, weak topics, hours studied
- 🔐 **Secure Auth** — OTP email verification + strong password + rate limiting
- 🌙 **3 Themes** — Dark, Light, Warm mode

## 🛠️ Tech Stack

| Layer                                    |            Technologies |
|------------------------------------------|----------------------------------------------|
| Backend                                  | Python, Django 6, SQLite                     |
| AI                                       | Google Gemini API                            |
| Frontend                                 | Bootstrap 5, Tailwind CSS, JavaScript        |
| Security                                 | OTP, Rate Limiting, CSRF, Session Management |
